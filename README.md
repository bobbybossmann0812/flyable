# flyable
Dieses Programm startet ein Programm, sobald der msfs2020 Flug geladen ist.

Ich hatte bislang das Problem, dass mein FSCOM von CPflight mein MCP nicht erkannt hat, da ich es erst starten muss, wenn der Flug wirklich geladen hat. Dieses Programm erleichtert diesen Schritt des nicht-vergessens.

## Requirements

ggfs Python installiert
order als standalone über flyable.exe.
